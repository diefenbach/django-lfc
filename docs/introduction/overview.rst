========
Overview
========

Goal
====

LFC aims to be easy and fast for both: users and developers.

Features
=========

**Users**

* Commenting
* Cut/Copy'n Paste
* Easy upload of images and files
* Multilingual content and GUI
* RSS Feeds
* Search
* Tagging
* Variable Templates
* Variable Portlets
* WYSIWYG-Editor
  
**Developers**

* Add own templates
* Add own portlets
* Add own applications and content types
