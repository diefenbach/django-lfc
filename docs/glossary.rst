Glossary
========

.. glossary::

  Portlet
    Portlets are pieces of content which are managed in Slots. Slots can be
    freely defined and placed anywhere within a HTML page.

    LFC has two slots: a left and a right slot beside the content in the
    middle of the page.

    For more information see `django-portlets <http://packages.python.org/django-portlets/>`_
