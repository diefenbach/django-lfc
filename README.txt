What is it?
===========

LFC is a Content Manangement System based on Django

Features:
=========

- Variable templates
- Variable portlets
- Multi languages (without pain)
- Bulk upload of image
- Automatic scaling of images
- WYSIWYG-Editor
- Tagging
- Commenting
- RSS Feeds
- Pluggable (Own content types and portlets)

Documentation
=============

For more documentation please visit: http://packages.python.org/django-lfc/
